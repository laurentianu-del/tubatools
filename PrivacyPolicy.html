隐私政策

图吧工具箱winui3 — 最后更新：2026年5月

数据收集

图吧工具箱winui3 不收集、不存储、不传输任何用户个人数据。

  • 不收集姓名、邮箱、电话号码等个人身份信息
  • 不收集设备标识符、IP地址或地理位置
  • 不使用Cookie或任何追踪技术
  • 不接入任何第三方分析、广告或社交SDK

本地数据存储

应用仅在您的设备本地存储以下数据：

  • 收藏夹数据 — 您标记为"收藏"的工具路径列表，存储在Windows本地应用设置中（ApplicationData.LocalSettings）
  • 图标缓存 — 工具可执行文件的图标提取缓存，存储在 %LocalAppData%\TubaWinUi3\IconCache\ 目录下

以上所有数据均保存在您的本地设备上，不会上传至任何服务器。卸载应用时，这些数据将被一并清除。

网络访问

本应用为纯离线应用，不进行任何网络通信。应用不会连接互联网，不会发送任何请求，也不会接收任何远程数据。

所需权限

应用声明以下权限用于核心功能：

  • runFullTrust — 用于启动第三方诊断工具、读取WMI硬件信息、编辑系统Hosts文件、管理证书存储、清理系统垃圾文件等系统级操作

第三方工具

本应用集成了多个第三方PC硬件诊断工具。这些工具作为独立程序运行，各有其自身的功能和数据处理方式。本应用仅提供统一的快捷启动入口，不干涉、不监控这些工具的数据行为。

儿童隐私

本应用不面向13岁以下儿童，也不会有意收集儿童的个人信息。

政策变更

如本隐私政策有任何变更，将在应用内公布。建议定期查阅本页面。

Privacy Policy

Tuba Toolbox WinUI3 — Last updated: May 2026

Data Collection

Tuba Toolbox WinUI3 does not collect, store, or transmit any user personal data.

  • No personal identity information (name, email, phone number, etc.)
  • No device identifiers, IP addresses, or geolocation
  • No cookies or tracking technologies
  • No third-party analytics, advertising, or social SDKs

Local Data Storage

The app stores only the following data locally on your device:

  • Favorites Data — A list of tool paths you have favorited, stored in Windows local application settings (ApplicationData.LocalSettings)
  • Icon Cache — Extracted icons from tool executables, stored under %LocalAppData%\TubaWinUi3\IconCache\

All data is stored exclusively on your local device and is not uploaded to any server. This data is removed when you uninstall the application.

Network Access

This application is fully offline and performs no network communication. It does not connect to the internet, send any requests, or receive any remote data.

Required Permissions

The app declares the following capabilities for core functionality:

  • runFullTrust — Required for launching third-party diagnostic tools, querying WMI hardware information, editing the system Hosts file, managing certificate stores, cleaning system junk files, and other system-level operations

Third-Party Tools

This application integrates multiple third-party PC hardware diagnostic tools. These tools run as independent programs with their own functionality and data handling. This application only provides a unified launcher and does not interfere with or monitor the data behavior of these tools.

Children's Privacy

This application is not directed to children under 13 and does not knowingly collect personal information from children.

Policy Changes

Any changes to this privacy policy will be announced within the application. Please check this page periodically.
